107年研究所甄試心得

稍微分享一下我的 107 年研究所推甄的成果和心得，希望學弟妹們看了可以有個依據，讓你們知道大概自己的成績可以推到哪一間研究所。


先說明一下我的推甄資料

  • 就讀學校： 逢甲大學
  • 科系： 通訊工程學系
  • 在校成績： 班排名(5/53)、系排名(10/106)
  • 校外成果： IMP2017論文接受並去研討會發表


推甄的校系

  • 成功大學電腦與通訊工程所甲組 (初審未通過)
  • 成功大學資訊工程所 (初審未通過)
  • 清華大學通訊所乙組 (初審未通過)
  • 中央大學資訊工程所 (初審通過)
  • 中山大學電機工程所丙組 (初審通過)
  • 逢甲大學通訊工程所 (直接錄取)


最後結果

  • 中央大學資訊工程所 (果斷放棄不去面試)
  • 中山大學電機工程所丙組 (正取)
  • 逢甲大學通訊工程所 (正取1)


中山大學 電機所 丙組 面試心得

  • 穿著： 當天幾乎男生都穿西裝、女生都穿套裝，所以建議至少男生也要穿個白襯衫+西裝褲，才不會顯得太過突兀，西裝外套的話就自己斟酌吧~可穿可不穿。

  • 面試方式： 一次進去都是 4 個人，而裡面有 4 個教授，每個教授都會單獨面試你 5 分鐘，所以總共面試 20 分鐘

  • 面試問題如下

    • 第一位教授： 主要都在詢問我的專題內容，還有我之後想要研究的方向

    • 第二位教授： 5 分鐘都在問我的專題內容

    • 第三位教授： 問我什麼是線性獨立以及機率獨立(我都沒回答出來~冏)，因為我的簡歷上有寫我會影像處理，所以還有問我影像處理的相關問題，例如像是物件追蹤、人臉辨識要用什麼演算法之類的

    • 第四位教授： 跟我哈拉了 5 分鐘


結論

總結來說，這次推甄的非常慘，我建議讀通訊的人不要輕易跨到資工領域，除非你在大學時期都有在修資訊系的課程，或者你在校外有很多資訊類的優異表現，否則推上的機率真的非常的低

一些基本的數學觀念要知道，例如線性獨立、機率獨立等等的。悲慘的是我讀了 3 年的書卻連這種觀念都不會，確實有點糟糕 ; 此外，自己的專題一定要完全清楚，因為問專題的時間佔了大多數

我面試當天很多人都有自己準備書面資料要給教授，但是一個教授也才 5 分鐘，根本不可能會看，所以建議大家把重要資料整理成一張 A4 的簡歷就好

在我面試完之後才去查了中山電機丙組的評價，發現非常的慘烈，我在放榜之後也有去中山找教授，如果真的想念電機丙的話，請你要做好至少 3 年的準備XD，所以我打算要延畢重考了QQ

如果有其他問題都可以來 我的 Facebook 私訊我，想要參考我的備審也可以來密我啦XD

解釋臉部偵測程式

其實過程很簡單，大致是以下幾個步驟：

  • 將人臉及人眼分類器演算法的 XML 檔 Load 進來 (程式第28~32行)
  • 使用 cvtColor() 將影像轉成灰階(GaryScale)，目的是減少運算量 (程式第66行)
  • 使用 resize() 更改影像大小比率，視情況而定，在這裡我們先不使用 (程式第70行)
  • 使用 equalizeHist() 將影像做直方圖等化，以增強對比度 (程式第74行)
  • 運用 XML 內的分類演算法來偵測出人臉和人眼 (程式第78行和91行)
